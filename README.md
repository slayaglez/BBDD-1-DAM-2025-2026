# BBDD-1-DAM-2025-2026
Repositorio creado para la asignatura de base de datos de 1º DAM.

### Despiece del repositorio
En las carpetas ***"Actividades"*** habrá un fichero de texto que contendrá lo siguiente:
-El enunciado del ejercicio
-El apartado donde se crean las tablas
-El apartado donde se insertan los datos
-El apartado donde se proponen consultas para resolver el enunciado

### Actividad 1
Ejercicio que consiste en la creación de una base de datos para un cliente que quiere saber la información de producto de sus competidores así como calcular sobre el precio de estos cosas como medias, medianas, modas, máximos y mínimos.
Fecha: 01/12/2025
